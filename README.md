# TroyMD Hub

Public-facing dashboard for the maintained TroyMD portfolio links. This is the active public routing hub for the portfolio and is distinct from the minimal `troyfowlermd.github.io` root-site repo.

## Link Policy

All dashboard destination links should open in a new tab by default with `target="_blank"` and `rel="noopener"` unless a future change explicitly calls for same-tab navigation. In-page section jump links may stay same-tab.

## Project Files

- `CONTEXT.md`: Short session-start briefing for Codex and returning developers.
- `TASKS.md`: Live working task list seeded from the migrated Notion state.
- `WORKLOG.md`: Append-only session-end worklog format.
- `DECISIONS.md`: Key routing and workflow decisions extracted from Notion.
- `docs/troymd-personal-dashboard.md`: Migrated TroyMD Personal Dashboard project context.
