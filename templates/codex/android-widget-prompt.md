# Android Widget Prompt

Codex prompt template for building Android Home Screen widgets that bridge to web apps via Capacitor and SharedPreferences.

---

Repo: github.com/TroyFowlerMD/non-clinical.
Goal: Create an Android Home Screen widget that displays [Data Key] and bridges to the main web app via Capacitor/SharedPreferences.
Constraints:
1. Use native Android/Kotlin for the widget provider.
2. Bridge the web-to-native data sync using SharedPreferences.
3. Ensure the widget is resizable and handles configuration changes.
4. Implement a clean view switcher for the widget UI.
